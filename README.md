# Mi Proyecto - React + Vite

Proyecto de React con Vite, React Router y ESLint.

## 📋 Requisitos Previos

Antes de instalar este proyecto, necesitas tener:
- **Node.js** (versión 14 o superior) - [Descargar](https://nodejs.org/)
- **npm** (viene incluido con Node.js)

## 🚀 Instalación

1. **Clonar o descargar** este repositorio
   ```bash
   git clone <tu-repositorio>
   cd mi-proyecto
   ```

2. **Instalar dependencias**
   ```bash
   npm install
   ```

## 💻 Comandos Disponibles

- **Modo desarrollo** (activa Vite con Hot Module Replacement)
  ```bash
  npm run dev
  ```

- **Compilar para producción**
  ```bash
  npm run build
  ```

- **Verificar el código con ESLint**
  ```bash
  npm run lint
  ```

- **Ver la compilación en el navegador**
  ```bash
  npm run preview
  ```

## 📦 Dependencias Principales

- **React** - Librería para interfaces
- **React DOM** - Renderizado en navegador
- **React Router DOM** - Enrutamiento entre páginas
- **Vite** - Herramienta de compilación rápida
- **ESLint** - Validador de código

## 🔧 Estructura del Proyecto

```
src/
├── components/     # Componentes reutilizables
├── pages/         # Páginas de la aplicación
├── services/      # Servicios de API/datos
├── models/        # Modelos de datos
└── utils/         # Funciones auxiliares
```
